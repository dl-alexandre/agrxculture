4. Typography Scaling

This text should scale appropriately for different screen sizes. On low-end mobile devices (≤479px), the font size should be smaller to accommodate limited screen space while maintaining readability.

The line height and spacing should also adjust to provide optimal reading experience across all device types commonly used in rural agricultural areas.

5. Container Responsive Behavior

The container should use appropriate padding and margins for each breakpoint:

  • Low-end mobile (≤479px): Minimal padding to maximize content area
  • Standard mobile (480-767px): Comfortable padding for touch interaction
  • Tablet (768-1023px): Increased padding for better visual hierarchy
  • Desktop (≥1024px): Maximum width with centered content
